# Toplax Static Website

Cleaned static HTML template structure.

## Project Structure

- `index.html` - main homepage
- `*.html` - other website pages
- `assets/css/` - Bootstrap, plugins, custom styles, local font CSS
- `assets/js/` - JavaScript libraries and template scripts
- `assets/images/` - site images, icons, logos, and backgrounds
- `assets/videos/` - video files used by the template
- `assets/webfonts/` - Font Awesome webfonts
- `assets/fonts/manrope/` - local Manrope font files
- `_reference/` - original web copy/reference metadata

## Run

Open `index.html` directly in a browser, or serve this folder from any static hosting/server.
